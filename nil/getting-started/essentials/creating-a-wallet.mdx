# Creating a wallet

This tutorial outlines the different ways for creating a wallet on =nil;.

## Via the RPC API

There are three possible flows for creating a wallet using the =nil; RPC API, all of which rely on calling various methods inside the faucet contract.

The faucet contract is deployed at addresss `0x000100000000000000000000000000000FA00cE7`.
